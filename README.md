# Mental Health Awareness Website

A simple, accessible one-page website designed to raise awareness about mental health.  
This project was built using **HTML**, **CSS**, and **Bootstrap 5**.

##Features
- Clean, responsive layout
- Navigation links that scroll to each section
- Sections include:
  - Home (Hero)
  - About Mental Health
  - Common Issues
  - Tips for Wellbeing
  - Resources
  - Affirmations
- Light teal theme for a calm, supportive feel
- Fully responsive on mobile, tablet, and desktop

##Technologies Used
- HTML5
- CSS3
- Bootstrap 5

## How to Run the Project
1. Download or clone the repository.
2. Open `index.html` in any web browser.

##Deployment
This site is deployed using GitHub Pages.
